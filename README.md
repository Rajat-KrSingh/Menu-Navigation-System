# Menu Navigation System

A DSA Mini Project built using Java.

## Technologies

- Java
- Tree Data Structure
- Java Servlet
- JSP
- HTML
- CSS
- Apache Tomcat

## Features

- Add Menu
- Delete Menu
- Rename Menu
- Search Menu
- Display Menu Tree
- Count Total Menus
- Calculate Tree Height

## Data Structure Used

Tree (General Tree)
